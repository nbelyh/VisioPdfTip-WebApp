# Chipher File Tool

This tool allows you to chipher the content of the Visio diagram. 
It analyzes the Visio file and chiphers all text and shape content, according to the selected chiphering options.
You can then save the chiphered file to a new location.

Your files will not leave your browser, all the processing will be done locally on your machine in this browser using web assemtly

Please report issues to our [GitHub Issue Tracker](https://github.com/nbelyh/VisioWebTools/issues)
